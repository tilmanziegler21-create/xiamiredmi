# 🎯 ДЕМОНСТРАЦИЯ БРЕНДОВЫХ ИЗОБРАЖЕНИЙ

## 📱 Как это выглядит в приложении:

### Вариант 1: Продукт с брендом ELFliq (без указания image)
```json
{
  "id": "SKU001",
  "name": "ELFliq Классик 60мл",
  "category": "liquids", 
  "brand": "ELFliq",
  "price": 18,
  "image": "", // Пустое поле
  "description": "Классический табачный вкус"
}
```
**Результат**: Показывается красивая брендовая картинка `/images/brands/elfliq/elfliq_liquid.png`

### Вариант 2: Продукт с конкретным изображением
```json
{
  "id": "SKU002", 
  "name": "Жидкость Премиум 60мл",
  "category": "liquids",
  "brand": "Premium",
  "price": 25,
  "image": "/images/products/custom-liquid.jpg", // Конкретное изображение
  "description": "Премиум вкус"
}
```
**Результат**: Показывается указанное изображение

### Вариант 3: Продукт без бренда и изображения
```json
{
  "id": "SKU003",
  "name": "Безбрендовая жидкость 60мл", 
  "category": "liquids",
  "brand": "", // Пустое поле
  "price": 15,
  "image": "", // Пустое поле
  "description": "Бюджетный вариант"
}
```
**Результат**: Показывается градиентный фон без изображения

## 🚀 Как протестировать:

1. **Откройте**: http://localhost:5174
2. **Выберите город** в выпадающем списке
3. **Перейдите в каталог** жидкостей
4. **Добавьте в Google Sheets** товар с брендом "ELFliq" (без поля image)
5. **Обновите страницу** и увидите красивую брендовую картинку!
