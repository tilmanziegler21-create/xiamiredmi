# E2E тесты Sheets

1) Заказ → резерв → не подтвердил → резерв снялся → stock не изменился
- Создать заказ, не выбирать слот, дождаться 15 минут → статус expired
- Проверить stock в Sheets не изменился

2) Подтверждение → delivered → stock уменьшился
- Создать заказ, выбрать курьера и слот, курьер «Выдача»
- Проверить stock в Sheets списался по позициям

3) Двойная «Выдача»
- Нажать «Выдача #ID» дважды
- Проверить stock списался один раз; orders.sheets_committed=true

4) Sheets недоступен → догонка
- Симулировать ошибку при коммите (отключить доступ)
- Статус delivered остаётся; sheets_committed=false
- Включить доступ; дождаться repair cron → stock списан один раз

5) Смена интервала курьера в Sheets
- Изменить interval в Sheets
- Через TTL бот показывает новый интервал
